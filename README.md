# Easy Sit

## How it works

**Right click mode**

1. Look all the way down
   
2. Sneak and unsneak
   
3. Press right click

#

**Double sneak mode**

1. Look all the way down
   
2. Double sneak

#

**To not sit any longer press shift**


## Config

**Change the way how to sit**

```
/scoreboard players set &sit_mode config.easysit value
```

value = 0 = Right click mode --> default

value = 1 = Double sneak mode
